FAQ

Quick answers about products, downloads, and orders.

Do the product downloads work offline?

Yes. The Word documents are stored in the docs/ folder and downloaded using normal file links.

Is this a real ecommerce checkout?

This is a demo flow for academic use. Cart and checkout state are stored in localStorage only.

Can I add more products?

Yes. Edit js/data.js and add new product objects. All pages update automatically.
